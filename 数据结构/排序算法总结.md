1. 插入排序    

	**描述**：  
		把待排序的元素插入到已经排好序的有序序列中。  
	**代码**：
	
			int[] arr = new int[1,3,5,2,6];
			for(int i = 1; i < arr.length; i++) {
				int temp = a[i];
				for(int j = i; j > 0; j--) {
					if(temp < a[j-1]){
						a[j] = a[j-1];
					} else {
						a[j] = temp;
					}
				}
			}
	**时间复杂度**：
	
		 * 最好情况(已经排好序)：O(N)
		 * 最坏情况（倒序）：N(1+N)/2==>O(N^2)
	    
		
	**适用情况**：  
	* 当数据基本有序的时候效率比较高
