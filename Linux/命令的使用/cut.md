##  cut 命令 

分割一行文字。

### 用法  

1. 分割一行取分割的第2，3个元素 

		# 取每行的第2，3个元素
        echo ${PATH} | cut -d ':' -f 2,3
		# 取倒数第一个元素
		last | cut -d ' ' -f -1
2. 取第几个字符之后的字符。

		# 取第三个之后的元素，包含第三个
		last | cut -c 3-
		# 取前三个元素
		last | cut -c -3
		# 取第一到三个元素 [1-3]
		last | cut -c 1-3
