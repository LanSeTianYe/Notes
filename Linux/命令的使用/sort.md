## sort  

### 选项  

* `-n`: 以数字的格式进行排序
* `-r`：倒序。
* `-f`：忽略大小写。
* `-b`：忽略前面空白字符。
* `-u`：去重，相同的内容只显示一行

### 用法  

1. 以 `:` 分割的第三个元素进行排序。

		cat /etc/passwd | sort -t ':' -k 3
2. 以 `:` 分割的第三个元素以数字的格式进行排序。

		cat /etc/passwd | sort -n -t ':' -k 3

3. 以 `:` 分割的第三个元素以数字的格式进行倒序排序。

		cat /etc/passwd | sort -rn -t ':' -k 3

4. 数组效果 

		cat /etc/passwd | cut -d ':' -f 3 | sort -n
