## 忽略项目的文件

### 创建忽略文件
在 `Git BASH` 下 创建 `.gitingore`文件

	touch .gitingore
### 忽略一些文件

	## 忽略HBuider工程文件
	*.progect
	## 忽略Intelij idea工程文件夹
	.idea/
	## 忽略Intellij idea 工程文件
	*.iml
	## 忽略日志文件
	*.log
	## 忽略maven生成的目标文件
	target/
	
### 忽略文件不起作用
如果某些文件已经被纳入了版本管理中，则修改.gitignore是无效的。那么解决方法就是先把本地缓存删除（改变成未track状态），然后再提交：

	git rm -r --cached .
	git add .
	git commit -m 'update .gitignore'
