## Date 

### 简介

显示或设置系统时间。

### 语法 

```shell
date [option]... [formort]
```

**选项：**

* `-d`： 字符串格式显示时间 `Fri Feb  6 11:32:03 CST 1970` 。 
	* `date -d @3123123`
	* `date -d @3123123 %s`
* `-r`: 显示文件的最后变更时间。
	* `date -r dump.rdb`
* `-u`: 显示或设置系统时间（UTC）。
	* `date -u`
* `-R`: 显示时区
    * `date -R`

**格式：**

* `%%`: 输出 `%`
* `%Y`: 年
* `%m`: 月（两位 1-12）
* `%d`：天（两位）
* `%H`: 小时（24）
* `%I`: 小时(12 两位)
* `%M`: 分（两位）
* `%S`: 秒（两位）
* `%s`: 自标准时间开始过了多少秒
* `%N`: 当前时间纳秒

## 例子 

1. 格式化年月日  

    ```shell
    date +'%Y-%m-%d %H-%M-%S %N'
    # 2018-11-12 11-38-10 714744478
    ```
2. 格式化时间并输出 

    ```shell
    date -d @12312312 +'%Y-%m-%d %H-%M-%S-%N'
    # 1970-05-23 20-05-12-000000000
    ```
3. 设置系统时间

    ```shell
    date -s "2020-07-11 18:08:08"
    ```
