## git模型
  
1. 工作区：存储修改过的未缓存的文件
2. 缓存区：缓存修改过的文件
3. 本地仓库：提交缓存文件到本地仓库
4. 远程仓库：推送本地仓库文件到远程仓库

工作流程：本地编辑或删除文件之后文件会放在工作区，`git add` 命令会把指定的文件放进缓存区，`git commit` 会把缓存区的文件提交到本地仓库。 `git push` 命令会把本地仓库中的文件推送到远程仓库。
