## Git rm  

删除文件并把文件放入缓存区，或把删除的文件放入缓存区。

## 常用操作 

1. 删除文件并把删除信息放入缓存区。
		
		git rm <file>...  

2. 把已经删除的文件放入缓存区。

		git rm <file>...

3. 删除文件，删除的信息放在缓存区，并把删除的文件放入工作区。

		git rm --cached <file>...

4. 删除文件，并停止跟踪文件，可用于想要忽略已经添加到仓库中的文件。
		
		# 停止追踪文件并删除
		git rm --cached <file>...
		# 添加文件到忽略文件
		.gitignore
